Les règles du jeu

Les règles sont les suivantes : si vous pariez sur la bonne équipe, vous remportez 1 point. Si vous pariez sur la bonne équipe et le ou les joueurs buteur(s), vous remportez 2 poins.
Si vous parvenez à trouver le score exacte ainsi que l'équipe gagnante, vous gagnez 3 points. Enfin, si vous choisissez l'équipe gagnante mais également le ou les buteur(s) et le score exacte, alors vous gagnez 5 points.
